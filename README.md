## 📋 프로젝트 개요

JavaPresso는 1차 CLI 기반 카페 관리 시스템을 웹 환경으로 전환한 프로젝트입니다. JSP/Servlet 기반의 웹 애플리케이션으로 개발되었으며, MVC 패턴을 적용하여 요구사항을 모두 충족하면서 그동안 배운내용을 충분히 이해하고 적용하는것을 목표로 개발했습니다.

### 🎯 프로젝트 목표

- CLI 기반 1차 프로젝트를 웹 기반으로 전환
- 사용자 친화적 인터페이스 구현 및 기능 확장
- MVC 패턴을 활용한 체계적인 웹 애플리케이션 개발

- **웹 기반 전환 방식**:
  - DB 구조 및 DAO 로직 유지

## 🚀 실행 방법

1. **환경 준비**

   ```bash
   # Docker 환경에서 Oracle 및 Tomcat 컨테이너 실행
   docker-compose up -d
   ```

2. **데이터베이스 설정**

   - Oracle DB 접속 후 필요한 테이블 생성
   - 초기 데이터 삽입

3. **프로젝트 배포**

   - WAR 파일을 Tomcat webapps 디렉토리에 배포
   - 또는 Eclipse에서 직접 Tomcat 서버에 배포

4. **접속**
   - 브라우저에서 `http://192.168.0.11:8080/` 접속
